OBS: Estilo CSS Inline, náo é recomendando devido a poluição de códigos dentro do HTML, sem valor semântico. Onde damos estilos a fonte, alinhamento, cor .... dentro de cada tag. Isso seria muito mais trabalhoso e desorganizado profissionamente falando. style="color(cor da fonte), "text-align: justify;" (parágrafo alinhado), background-color(cor do fundo, usado dentro da tag <body>), font-family(tipo da fonte)

Código do texto acima: 

            
                <body style="background-color: aqua;font-family: Arial, Helvetica, sans-serif;">
                    <h1 style="color: red;">Capitulo1</h1>
                    <h2 style="color: blue;">Capitulo1.1</h2>
