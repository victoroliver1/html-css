ANOTAÇOES

OBS: Estilos Css Externo são mais versáteis de ser usado. Primeiro passo é criar um arquivo Css. Em Head abaixo do tittle devemos criar um link css através da tag link:css, exemplo: <link rel="stylesheet" href="style.css">.


Segundo passo é transferir ou criar todo estilo dentro do arquivo CSS, dessa forma o estilo ficará registrado em todo o projeto, náo se restringindo apenas a uma página como no estilo interno, como podemos vê na página02.

Podemos misturar o estilo inline (não recomendado, usado apenas em mudanças pontuais), com o estilo interno (usado apenas para colocar estilo em uma página) e o estilo externo (usado para aplicar o mesmo estilo a todo projeto).
