Anotaçoes

  • a parâmetro id"" é usado para designar um estilo CSS externo específico para uma tag.
  • Tudo o que usamos em HTML como id = em CSS é: #
  • Tudo o que usamos em CSS como class = em CSS é: .
  • Segundo a W3schools, é permitido apenas um id por documento html.
  • É permitido dá varias class por elemento.
  • É permitido dá mais de uma class no mesmo elemento, exemplo: <h2 class="avancado destaque">.CSS Avançado</h2> (sem usar vírgula, apenas espaço).

    • Exemplo código com id HTML E CSS:
    No HTML: <h1 id="principal">Criando Sites com HTML e CSS</h1>
    No CSS: 

                
                h1#principal{
                    text-align: center;
                    background-color:  rgb(74, 136, 115);
                    color: white;}
    OBS: Neste exemplo aplicamos o nome de id="principal" ao h1 no HTML e aplicamos o estilo próprio para ele no CSS usando o chaveamento h1#principal{}. Texto centralizado, cor do fundo e cor da fonte (text-align, background-color e color)

    Exemplo código com classHTML e CSS:
    No HTML:<h2 class="intermediario"> CSS Intermediário</h2>
    No CSS:.

                .intermediario{
                color:  rgb(89, 95, 8);
            }
    OBS: Neste exemplo aplicamos uma class"intermediario" no h2 pelo HTML e aplicamos o estilo no CSS usando o chaveamento .intermediario{ color: rgb(89, 95, 8); }
