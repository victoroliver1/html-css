Usando links externos

Você pode acessar meu repositório público no GitHub

Você tabmém pode acessar o meu canal gratuito no YouTube

OBS: Para utilizar links externos basta adicionar a tag <a> e dentro do href adicionar a URL completo para qual deseja o redirecionamento do link. De extrema importância utilizar o parâmetro target="_blank" rel="external" dentro da tag <a> para que o site externo seja redicionado para outra aba. Por exemplo: 

<a href="URL DO SITE" target="_blank" rel="external>"

OBS: Para adicionar um link para download basta criar uma lista usando a tag <ul> e dentro da tag <li> adicionar a tag <a> com os seguintes parâmentros: <a href="local onde está o arquivo" download="nome do arquivo" type="application/tipo do arquivo">.Se o arquivo for em pdf por exemplo o parâmetro ficaria: application/pdf. O "application" para cada tipo de arquivo pode ser encontrado no site https://www.iana.org/assignments/media-types/.
