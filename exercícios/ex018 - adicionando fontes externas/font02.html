Adicionando Fontes Externas

1º passo é baixar a fonte e colar dentro da pasta do vscode (recomendado criar uma pasta font)

2º passo é adicionar o seguinte código dentro da tag <style>: 

             
                @font-face {
                    font-family: 'nome da fonte'  ;
                    src: url(caminho da fonte) format('nome do formato');
                }

OBS:

  • Deixar sempre entre parenteses o 'nome da fonte' e 'nome do formato'
  • As fontes geralmente virão em formato opentype(otf) e truetype(ttf) que são compatíveis com todos os navegadores e dispositivos.

    • Exemplo de código: 

                
                @font-face {
                    font-family: 'Love larry'  ;
                    src: url(fonts/love\ larry\ ttf.ttf) format('truetype');
                }

  • Usar a extensão FontsNinja para identificar qualquer fonte de qualquer site.
