OBS: Podemos através dos estilos internos organizar e facilitar o estilo do site de forma mais organizada. Pra isso abrimos uma tag <style> dentro da Head abaixo do tittle e ao invés de abrir tags com <>, abrimos com {}. Uma vez aplicado o estilo em uma tag, será aplicada a todas as tags da página. Exemplo do código do texto acima.


    
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Estilos Locais / Internos</title>
            <style>
                body{
                    background-color: lightblue;
                    font-family: Arial, Helvetica, sans-serif;
                    font-size: 20px;
                }
                h1{
                    color: darkblue;
                    background-color: lightskyblue;
                }
                h2{
                    color: darkred;
                }
                p{
                    text-align: justify;
                }
            </style>
        </head>
    
    
        </body>


Como exemplo no: 

    
        p{
            text-align: justify;
        }
todas as tags <p>, estarão com o texto alinhado(text-align: justify), assim como as demais tags aplicando o estilo no head, chamado de estilos internos ou locais.
